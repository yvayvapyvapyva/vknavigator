🧭 NAVIGATOR — приложение для навигации по учебным маршрутам с голосовыми подсказками

════════════════════════════════════
📱 ГЛАВНЫЙ ЭКРАН (НАВИГАТОР)
════════════════════════════════════


ЭЛЕМЕНТЫ УПРАВЛЕНИЯ:

☰ МЕНЮ (вверху по центру)
   Открывает боковое меню с вкладками:
   • Ваши маршруты — список личных маршрутов
   • Каталог — публичные маршруты для тестирования
   • Инструкция — этот текст

◄ СПИСОК ТОЧЕК (слева от меню)
   Показывает все точки текущего маршрута
   Можно выбрать любую точку для просмотра

► ГЕОЛОКАЦИЯ (справа от меню)
   Привязка карты к вашему местоположению
   🔴 Активно — карта следует за вами
   ⚪ Выключено — карта зафиксирована

ТЕЛЕМЕТРИЯ (под кнопкой геолокации)
   • КМ/Ч — текущая скорость
   • АЗ — угол направления движения (азимут)

ИНФОРМАЦИОННАЯ ПАНЕЛЬ (внизу экрана)
   • № — номер текущей точки
   • Команда — голосовая команда точки
   • Дист — расстояние до точки (зелёный = близко)
   • АЗ — целевой азимут точки (зелёный = совпадает)
   • 🔊 — воспроизвести команду вручную
   • ◄ ► — переключение между точками

════════════════════════════════════
🎯 КАК РАБОТАЕТ НАВИГАТОР
════════════════════════════════════

1. Выберите маршрут в меню «Ваши маршруты» или «Каталог» 
2. Навигатор загрузит маршрут и покажет точки на карте
3. При движении приложение отслеживает:
   • Расстояние до следующей точки
   • Совпадение направления движения с азимутом точки
4. При приближении (< 25 м) и совпадении азимута (< 45°)
   автоматически воспроизводится голосовая команда
5. После выполнения команды переход к следующей точке

💬 КОММЕНТАРИИ К ТОЧКАМ
   Если у точки есть комментарий (оранжевая иконка «i»),
   он покажется в информационной панели.
   Нажмите «×» чтобы скрыть комментарий.

👁️ ПРЕДПРОСМОТР ТОЧКИ
   • Нажмите на любую точку на карте для предпросмотра
   • Используйте кнопки ◄ ► для просмотра соседних точек
   • Нажмите «Выбрать» чтобы сделать точку текущей
   • Нажмите на карту чтобы закрыть предпросмотр

📝 РЕДАКТОР МАРШРУТОВ
════════════════════════════════════

СОЗДАНИЕ НОВОГО МАРШРУТА:
1. В меню нажмите «Создать новый маршрут»
2. Введите название (только a-z, 0-9, _)
3. Откроется редактор маршрутов

ДОБАВЛЕНИЕ ТОЧКИ:
1. Нажмите «Добавить новую точку» внизу экрана
2. В окне создания выберите:
   • Цвет (категория манёвра)
   • Голосовую команду из списка
   • Комментарий (необязательно)
3. Нажмите «Поставить на карту»
4. Коснитесь места на карте, где должна прозвучать команда
5. Нарисуйте путь манёвра последовательными нажатиями
6. Нажмите «Завершить рисование пути»

🛰️ ЗАПИСЬ МАРШРУТА ПО GPS
   В редакторе нажмите кнопку 🌍 (справа вверху)
   • Приложение автоматически записывает путь при движении
   • Минимальное расстояние между точками: 5 м
   • Нажмите «Завершить запись» когда закончите

✏️ РЕДАКТИРОВАНИЕ ТОЧЕК:
   • Нажмите на точку на карте для редактирования
   • Или откройте список точек (кнопка названия маршрута)
   • Можно изменить: цвет, команду, комментарий, путь
   • Кнопка «Изменить путь» — рисование нового пути
   • Кнопка «Удалить» — удалить точку из маршрута

💾 СОХРАНЕНИЕ:
   Нажмите «Сохранить изменения» вверху экрана
   Сохраняйте периодически во время редактирования!

⚠️ ТРЕБОВАНИЯ И ОГРАНИЧЕНИЯ
════════════════════════════════════

📍 ГЕОЛОКАЦИЯ
   Требуется для работы навигатора
   Разрешите доступ к местоположению

📱 ПОСТОЯННО ВКЛЮЧЁННЫЙ ЭКРАН
   Приложение пытается предотвратить отключение экрана,
   это ограничение веб-приложений

❌ ПРИ ВЫХОДЕ ИЗ ПРИЛОЖЕНИЯ:
   • Сворачивание браузера
   • Выключение экрана
   • Переход в другое приложение
   → Отслеживание геопозиции останавливается

🌐 ИНТЕРНЕТ
   Требуется для:
   • Загрузки карт Яндекс
   • Синхронизации с облаком (сохранение маршрутов)
   • Загрузки аудиокоманд

   При проблемах с подключением используйте VPN

════════════════════════════════════
💡 СОВЕТЫ
════════════════════════════════════

1. Перед поездкой сохраните маршрут в кэш (откройте в навигаторе)
2. Проверьте работу голосовых команд на первой точке
3. Держите экран включённым во время движения
4. При потере сигнала выберите следующую точку вручную
5. Используйте комментарии для заметок об особенностях манёвра
6. Для повторного прохождения маршрута используйте «Каталог»

════════════════════════════════════
🆘 ЕСЛИ ЧТО-ТО ПОШЛО НЕ ТАК
════════════════════════════════════

❌ Команда не сработала автоматически:
   → Нажмите 🔊 для ручного воспроизведения
   → Или выберите следующую точку в списке

❌ Карта не следует за местоположением:
   → Нажмите кнопку геолокации (справа от меню)
   → Проверьте разрешения браузера

❌ Маршрут не загружается:
   → Проверьте интернет-соединение
   → Попробуйте VPN при проблемах с Telegram

❌ Аудио не воспроизводится:
   → Коснитесь экрана для активации звука
