# VK Navigator — Навигатор маршрутов для VK Mini Apps

Веб-приложение для навигации по маршрутам с голосовыми командами, адаптированное для VK Mini Apps.

## 📋 Обзор

Приложение позволяет:
- **Каталог маршрутов** — публичные маршруты для тестирования
- **Пройти маршрут** — навигатор будет воспроизводить аудио-команды при приближении к точкам
- **Хранение данных** — GitHub Gists API (публичные маршруты)

## 🚀 Развёртывание во VK Mini Apps

### Шаг 1: Хостинг приложения

Приложение должно быть размещено на HTTPS хостинге. Варианты:

- **GitHub Pages** (бесплатно)
- **Vercel** (бесплатно)
- **Netlify** (бесплатно)
- Ваш собственный HTTPS хостинг

#### Развёртывание на GitHub Pages:

1. Создайте репозиторий на GitHub
2. Загрузите все файлы проекта
3. В настройках репозитория включите GitHub Pages
4. Выберите ветку `main` и папку `/root`

### Шаг 2: Создание VK Mini App

1. Перейдите в [VK Developers](https://dev.vk.com/)
2. Создайте новое приложение типа "Mini App"
3. В настройках приложения укажите:
   - **URL приложения**: `https://<ваш-хостинг>/index.html`
   - **URL для мобильной версии**: `https://<ваш-хостинг>/index.html`

### Шаг 3: Настройка Яндекс.Карт

В файле `index.html` замените `YANDEX_API_KEY` на ваш ключ API Яндекс.Карт:

1. Получите ключ в [Кабинете разработчика Яндекс](https://developer.tech.yandex.ru/)
2. Выберите "JavaScript API и HTTP Геокодер"
3. Вставьте ключ вместо `YANDEX_API_KEY`

### Шаг 4: Настройка каталога маршрутов

Отредактируйте файл `katalog.txt` и добавьте ваши маршруты:

```json
{
    "7292595756-m_1": "Экзаменационный маршрут Мещера 1",
    "7292595756-Razvoroty_perek_1": "Развороты на перекрестках Мещера"
}
```

Формат: `"USER_ID-ROUTE_NAME": "Отображаемое название"`

Маршруты должны храниться в публичных GitHub Gists пользователя с ID `USER_ID`.

### Шаг 5: Ссылки для запуска

#### Для VK Mini Apps:

Ссылка для запуска навигатора с маршрутом:
```
https://vk.com/app<APP_ID>#?route=<USER_ID>-<ROUTE_NAME>
```

#### Параметры:
- `<APP_ID>` — ID вашего VK приложения
- `<USER_ID>` — ID пользователя VK (владелец маршрута)
- `<ROUTE_NAME>` — название маршрута (без .json)

## 🎮 Как использовать

### Запуск навигатора:

1. Откройте приложение во VK
2. Выберите маршрут из каталога
3. Нажмите на маршрут — запустится навигатор
4. Следуйте голосовым командам

### Навигация по маршруту:

- Приложение автоматически воспроизводит голосовые команды при приближении к точкам (25м)
- Телеметрия показывает текущую скорость и азимут
- Можно вручную переключаться между точками через список

## 🔊 Голосовые команды

Приложение использует 43 аудиофайла с командами:
- Манёвры на перекрёстках
- Развороты
- Повороты
- Остановки
- Парковки
- И другие

## 🛠 Технические детали

### Платформы:
- **VK Mini Apps** (основная)
- Telegram WebApp (резервная)
- Веб-браузеры (резервная)

### API:
- **VK Bridge** — интеграция с VK
- **GitHub Gists API** — хранение публичных маршрутов
- **Yandex Maps API** — карты и геолокация

### Особенности:
- Адаптивный дизайн для мобильных устройств
- Кэширование аудио в Service Worker Cache
- Wake Lock для предотвращения выключения экрана
- GPS трекирование в реальном времени
- Триггеры команд по расстоянию (25м) и азимуту

## 📞 Поддержка

При возникновении проблем:
1. Проверьте консоль браузера на ошибки
2. Убедитесь, что Яндекс.Карты API ключ активен
3. Проверьте, что VK приложение настроено правильно
4. Убедитесь, что маршруты в каталоге существуют в GitHub Gists
