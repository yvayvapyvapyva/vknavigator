# VK Navigator — Навигатор маршрутов для VK Mini Apps

Веб-приложение для навигации по маршрутам с голосовыми командами, адаптированное для VK Mini Apps.

## 📋 Обзор

Приложение позволяет:
- **Создавать маршруты** — ставить точки на карте, рисовать пути манёвров, назначать голосовые команды
- **Пройти маршрут** — навигатор будет воспроизводить аудио-команды при приближении к точкам
- **Каталог маршрутов** — публичные маршруты для тестирования
- **Хранение данных** — GitHub Gists API

## 🚀 Развёртывание во VK Mini Apps

### Шаг 1: Подготовка GitHub Gists

Приложение использует GitHub Gists для хранения маршрутов. Вам понадобится:

1. **GitHub аккаунт**
2. **GitHub Personal Access Token** с правами `gist`

Получение токена:
1. Перейдите на https://github.com/settings/tokens
2. Нажмите "Generate new token (classic)"
3. Выберите права: `gist`
4. Скопируйте токен

### Шаг 2: Хостинг приложения

Приложение должно быть размещено на HTTPS хостинге. Варианты:

- **GitHub Pages** (бесплатно)
- **Vercel** (бесплатно)
- **Netlify** (бесплатно)
- Ваш собственный HTTPS хостинг

#### Развёртывание на GitHub Pages:

1. Создайте репозиторий на GitHub
2. Загрузите все файлы проекта
3. В настройках репозитория включите GitHub Pages
4. Выберите ветку `main` и папку `/root`

### Шаг 3: Создание VK Mini App

1. Перейдите в [VK Developers](https://dev.vk.com/)
2. Создайте новое приложение типа "Mini App"
3. В настройках приложения укажите:
   - **URL приложения**: `https://<ваш-хостинг>/index.html`
   - **URL для мобильной версии**: `https://<ваш-хостинг>/index.html`

### Шаг 4: Настройка URL приложения

В файле `editor.js` найдите переменную `VK_APP_ID` и замените на ID вашего приложения:

```javascript
const VK_APP_ID = 'ваш_app_id';
```

ID приложения можно найти в настройках приложения на VK Developers.

### Шаг 5: Настройка Яндекс.Карт

В файлах `index.html` и `editor.html` замените `YANDEX_API_KEY` на ваш ключ API Яндекс.Карт:

1. Получите ключ в [Кабинете разработчика Яндекс](https://developer.tech.yandex.ru/)
2. Выберите "JavaScript API и HTTP Геокодер"
3. Вставьте ключ вместо `YANDEX_API_KEY`

### Шаг 6: Ссылки для запуска

#### Для VK Mini Apps:

Ссылка для запуска навигатора с маршрутом:
```
https://vk.com/app<APP_ID>#nav?route=<USER_ID>-<ROUTE_NAME>&t=<TOKEN>
```

Ссылка для запуска редактора:
```
https://vk.com/app<APP_ID>#editor?route=<ROUTE_NAME>&t=<TOKEN>
```

#### Параметры:
- `<APP_ID>` — ID вашего VK приложения
- `<USER_ID>` — ID пользователя VK
- `<ROUTE_NAME>` — название маршрута (без .json)
- `<TOKEN>` — первые 10 символов GitHub токена

## 🎮 Как использовать

### Создание маршрута:

1. Откройте приложение во VK
2. Нажмите "Создать новый маршрут"
3. Введите название (только a-z, 0-9, _)
4. Нажмите "Добавить новую точку"
5. Выберите категорию манёвра
6. Выберите голосовую команду
7. Поставьте точку на карте
8. Нарисуйте путь манёвра
9. Сохраните маршрут

### Запуск навигатора:

1. Выберите маршрут из списка
2. Нажмите "Запустить навигатор"
3. Следуйте голосовым командам

Приложение использует 43 аудиофайла с командами:
- Манёвры на перекрёстках
- Развороты
- Повороты
- Остановки
- Парковки
- И другие

## 🛠 Технические детали

### Платформы:
- **VK Mini Apps** (основная)
- Telegram WebApp (резервная)
- Веб-браузеры (резервная)

### API:
- **VK Bridge** — интеграция с VK
- **GitHub Gists API** — хранение маршрутов
- **Yandex Maps API** — карты и геолокация
- **Telegram Bot API** — отправка отчётов

### Особенности:
- Адаптивный дизайн для мобильных устройств
- Кэширование аудио в Service Worker Cache
- Wake Lock для предотвращения выключения экрана
- GPS трекирование в реальном времени
- Триггеры команд по расстоянию (25м) и азимуту

## 📝 Заметки

### Отличия от Telegram версии:

1. **SDK**: Telegram WebApp → VK Bridge
2. **Авторизация**: Telegram User → VK User
3. **Навигация**: `startapp` → `vk_ref` / `vk_launch`
4. **Ссылки**: `t.me/bot` → `vk.com/app<APP_ID>`

### Совместимость:

Приложение полностью обратно совместимо с Telegram версией. Все функции работают на обеих платформах.
